# 🔐 Password Strength Analyzer
A Python-based tool that evaluates the strength of user-entered passwords and provides feedback to improve password security.

## 📌 Project Overview
The Password Strength Analyzer checks passwords based on different security factors such as length, character types, and password complexity. It helps users understand password strength and follow better security practices.

## ✨ Features

- 🔍 Analyze password strength
- ✅ Checks password requirements:
  - Minimum password length
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- 📊 Displays password strength level:
  - Weak
  - Medium
  - Strong
- 💡 Provides suggestions to create stronger passwords
- 📝 Maintains password analysis history
